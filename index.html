liblangtag is an interface library to access/deal with tags for identifying languages, which is described in RFC 5646.

Features

  • several subtag registry database supports:
    • language
    • extlang
    • script
    • region
    • variant
    • extension
      • RFC6067
      • RFC6497
      • New T Extensions for language/locale identifier in CLDR 21.0.2
    • grandfathered
    • redundant
  • handling of the language tags
    • parser
    • matching
    • canonicalizing
